# Алгебра — основные концепции

## Линейные уравнения
Линейное уравнение имеет вид:
ax + b = 0

Решение:
x = -b / a

---

## Квадратные уравнения
Общий вид:
ax² + bx + c = 0

Решение через дискриминант:
D = b² - 4ac

Если:
- D > 0 → два корня
- D = 0 → один корень
- D < 0 → нет вещественных корней

---

## Матрицы
Матрица — это таблица чисел, используемая для линейных преобразований.

Применение:
- компьютерная графика
- нейросети
- системы уравнений
